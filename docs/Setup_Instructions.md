# Setup Instructions

## JDK Version
- JDK 17.0.10 (OpenJDK)

## Setup Process
1. Downloaded and installed JDK from Oracle's official website
2. Set JAVA_HOME environment variable
3. Added %JAVA_HOME%\bin to PATH
4. Verified installation with `java -version`
